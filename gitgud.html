What is the Command Line?

Text interface to an operating system.

It's 2018 Why Use Command Line?

  • Interact with non-GUI servers
  • Somethings are easier on command line

Where is the Command Line?

Mac: Command + Space Bar then type Terminal

Windows: Start Button -> type cmd and open "Command Prompt"

Print Working Directory

Mac: pwd

Windows: cd

Windows uses backslash separator while Mac uses forward slash

Display Manual Page

Mac: man

Windows: help

Flags and Switches

Mac: dash (-)

Windows: forward slash (/)

List files

Mac: ls

Windows: dir

Change Directory

Mac: cd

Windows: cd

Use Tab and always forget everything

Use Up arrow for command history

Environment Variables

Mac: env

Windows: set

$PATH and %PATH%

Which command am I using?

Mac: which

Windows: where

Mac EOF: Control + D

Windows EOF: Control + Z
